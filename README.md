# EduWeb – University Website UI

A responsive university website built using HTML, CSS, and JavaScript. This project showcases a modern and clean user interface suitable for educational institutions.


## Features

* Responsive design (mobile-friendly )
* Navigation menu with toggle functionality
* Hero section with call-to-action
* Courses section
* Campus showcase
* Facilities section
* Student testimonials
* Call-to-action banner
* Footer with social media icons


## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)
* Font Awesome Icons
